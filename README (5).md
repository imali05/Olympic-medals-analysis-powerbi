## 🎯 Project Objective

To create an interactive **Power BI dashboard** that effectively visualizes historical Olympic Games data, providing insights into:

- Total medals won per country (Summer vs. Winter)
- Medal count comparisons across nations
- Trends over time
- Distribution of Gold, Silver, and Bronze medals
- Participation vs. Success rate analysis

---

## 📊 Dashboard Overview

### Key Metrics
- **156 teams** participated across all Olympic Games
- **20,281 total medals** awarded from 1896 to 2020

### Dashboard Sections

| Section | Description |
|---|---|
| **Total Medals by Team** | Bar chart comparing Summer (purple) vs Winter (blue) medals per country |
| **Medal Count Comparison** | Side-by-side comparison of top nations |
| **Trends Over Time** | Historical medal trend lines for selected countries |
| **Medal Distribution by Type** | % breakdown of Gold / Silver / Bronze per country |
| **Participation vs. Success Rate** | Games participated vs. medals won analysis |

## 🚀 How to Run the Dashboard

### Prerequisites
- Install **Microsoft Power BI Desktop** (free):  
  🔗 [https://powerbi.microsoft.com/desktop](https://powerbi.microsoft.com/desktop)

### Steps

1. **Clone this repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/olympic-medals-analysis-powerbi.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd olympic-medals-analysis-powerbi
   ```

3. **Open the dashboard**
   - Double-click `Dissanayaka_Dashboard.pbix`  
   — OR —  
   - Open **Power BI Desktop** → `File` → `Open` → select `Dissanayaka_Dashboard.pbix`

4. **Explore the dashboard**
   - Use the interactive filters and slicers to explore different countries, years, and medal types.
